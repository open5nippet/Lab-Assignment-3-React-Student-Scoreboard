# 🎓 React Student Scoreboard

A sleek, high-performance **Student Scoreboard** application built with **React** and **Vite**. This application allows educators or administrators to manage student scores efficiently with a modern, responsive interface.

![React](https://img.shields.io/badge/React-20232A?style=for-the-badge&logo=react&logoColor=61DAFB)
![Vite](https://img.shields.io/badge/Vite-646CFF?style=for-the-badge&logo=vite&logoColor=white)
![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=for-the-badge&logo=css3&logoColor=white)

---

## 🚀 Features

- **✅ Student Management**: Add new students with names and initial scores effortlessly.
- **📈 Dynamic Score Updates**: Update student scores in real-time with instant feedback.
- **🗑️ Quick Delete**: Remove student records from the scoreboard with a single click.
- **📊 Live Statistics**: Automatically calculates and displays:
  - Total number of students.
  - Average score across the board.
  - Highest and lowest scores (optional/future enhancement).
- **📱 Responsive Design**: Fully optimized for various screen sizes, from desktops to mobile devices.
- **⚡ Fast Performance**: Powered by Vite for near-instant hot module replacement (HMR).

---

## 🛠️ Tech Stack

- **Frontend**: [React.js](https://reactjs.org/) (Hooks, Functional Components)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: Vanilla CSS (Custom professional theme)
- **Icons/Fonts**: Google Fonts & SVG Icons

## 📦 Installation & Setup

Follow these steps to get the project running locally:

### 1. Clone the Repository
```bash
git clone https://github.com/open5nippet/Lab-Assignment-3-React-Student-Scoreboard.git
cd Lab-Assignment-3-React-Student-Scoreboard
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Run Development Server
```bash
npm run dev
```
The application will be available at `http://localhost:5173`.
